# Projeto-Final-Programa-o-1-Arthur-Vieira-Anza
Projeto Final da cadeira de Programação 1

Para utilizar a interface de usuário, apenas inicialize App.java que ambas as telas serão abertas.
Após aberta, vá na interface Primárie e use os botões de "Adicionar ao pedido" para incluir um ou mais produtos do menu ao seu pedido.
Caso queira retirar um produto do pedido, clique no botão "Retirar ultimo item do pedido", para remover o item que foi colocado por último.
Ao terminar de escolher, clique em "FINALIZAR PEDIDO" para envia-lo para a interface Secundária (interface da cozinha).
